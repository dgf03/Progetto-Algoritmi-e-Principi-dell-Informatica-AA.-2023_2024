# Progetto di Algoritmi e Principi dell'Informatica
### :it: Progetto di Ingegneria Informatica per il corso "Algoritmi e Principi dell'Informatica" al Politecnico di Milano
*Anno accademico: 2023/2024*

*Votazione: 30/30 con lode*

**Riassunto**:

Implementazione di un programma C che simula il funzionamento a tempo discreto di una pasticceria industriale, specialmente la gestione degli ordini, del magazzino e del ricettario. Il programma deve usare gli algoritmi e le strutture dati appropriati in modo da garantire efficienza in termini di tempo e occupazione di memoria.

### :gb: Engineering of Computing Systems project for the course "Algorithms and Principles of Computer Science" at Politecnico di Milano
*Academic year: 2023/2024*

*Grade: 30/30 cum laude*

**Summary**:

Implementation of a C program to simulate a time-discrete operation of an industrial bakery, specially the managing of the orders, the stock and the recipe book. The program has to use the appropriate algorithms and data structures in order to ensure efficiency in terms of time and data storage.
